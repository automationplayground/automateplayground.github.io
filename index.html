Guide Finding Xpath (Easy mode)
Introduce 3 General Xpath Forms
  1. //tagname[@attribute='value']
  2. //tagname[text()='inner_html_text']
  3. //tagname[contains(@attribute or function, 'value')]
Get Xpath with Developer Tool

- Open developer tools by press "F12" or right click and select inspect. Then, the dev tool with Elements tab will display on your browser!

  1. Click this icon to activate inspector
  2. Move the cursor to element, then click
  3. HTML of the element will be highlighted
  4. Click anywhere on the Element tab to focus, then Press "Ctrl+F" or "Command + F", a search box will be displayed
  5. Select a Xpath form for the element
  6. Replace with its tagname, attribute name and value or proper value for others Xpath forms such as Text
